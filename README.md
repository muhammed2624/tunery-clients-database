# Automated Client Health Records System for Tunery Wellness Hub

**Role:** Administrative & Operations Officer
**Tool:** Microsoft Excel

## Overview

Tunery Wellness Hub had no organized way of keeping track of client information. Records were kept manually, on paper, which made it slow and unreliable to pull up a client's history or produce a clean report when one was needed. I built an Excel-based client database that stores each client's personal and health details and can generate a printable report sheet automatically, with no manual retyping involved.

## The Problem

Before this system, Tunery had no centralized client records. Information was scattered or handwritten, which meant staff had to search manually every time a client's details or history were needed, and there was no consistent, professional way to hand a client a printed summary of their information.

## What I Built

I designed a structured Excel database to capture each client's personal information alongside basic health details such as age, existing conditions, and allergies, plus their treatment and product history. A separate report sheet pulls from this database automatically, so staff can select a client and generate a ready-to-print report without touching the underlying data or retyping anything.

## How the Automation Works

- VLOOKUP and INDEX-MATCH formulas pull a selected client's information from the database into the report sheet automatically.
- Dropdowns and data validation on the input sheet keep entries consistent and reduce data entry errors.
- The report sheet itself is a print-ready template, formatted so it can be handed to a client or filed immediately after printing.

## Impact

Client records went from scattered paper notes to a single organized system. What used to require manually compiling a client's details now takes selecting a name from a dropdown and printing while cutting report preparation down to seconds and giving Tunery a consistent, professional document to give clients.

## Skills Demonstrated

Excel formulas (VLOOKUP, INDEX-MATCH), data validation and form design, database structuring, process automation, and print-ready document design.
